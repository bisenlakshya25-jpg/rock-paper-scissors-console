# 🎮 Rock Paper Scissors (Python)

A console-based Rock Paper Scissors game developed in Python as part of my Python project series. The game offers multiple game modes, input validation, score tracking, and a clean menu-driven interface.

---

## ✨ Features

- 🎮 Menu-driven interface
- 🤖 Play against the computer
- 🎲 Random computer moves
- ✅ Input validation
- 📊 Live score tracking
- 🔁 Play Again option
- 🏆 Best of 5 Mode
- 📈 Match statistics
- 🚪 Quit anytime using `Q`

---

## 🕹️ Game Modes

### Normal Mode
- Play unlimited rounds.
- Decide after every round whether to continue or end the match.
- Final match statistics are displayed at the end.

### Best of 5 Mode
- First player to win **3 rounds** wins the match.
- If no player reaches 3 wins after 5 rounds, the match ends and the final statistics are displayed.

## 📜 Game Rules

- 🪨 Rock beats Scissors
- ✂️ Scissors beat Paper
- 📄 Paper beats Rock
